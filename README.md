# 학생 역량 관리 시스템 (SCMS)

> Student Competency Management System

학생들의 비교과 활동, 역량 진단, 마일리지, 진로 설계, 상담을 통합 관리하는 웹 플랫폼입니다.

## 프로젝트 개요

### 주요 기능
- **비교과 프로그램**: 프로그램 등록, 조회, 신청, 참여 확인
- **역량 관리**: 역량 진단 테스트, 결과 분석
- **마일리지**: 적립, 조회, 역량 인증
- **진로 설계**: 진로 탐색, 포트폴리오, 채용공고 연동
- **통합 상담**: 심리상담, 온라인상담, 집단상담 예약
